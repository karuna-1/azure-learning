# Azure Fundamentals - Lesson 1

## Cloud Computing Basics

Cloud computing is the delivery of computing resources such as servers, storage, databases, and networking over the Internet instead of owning physical hardware.

### Why do companies use Cloud Computing?

- Pay only for what they use
- Scale resources up or down easily
- Avoid buying expensive physical servers

### Traditional Computing vs Cloud Computing

#### Traditional Computing

- Buy physical servers
- Maintain hardware yourself
- High upfront cost

#### Cloud Computing

- Rent servers from cloud provider
- Provider maintains hardware
- Pay only for what you use
