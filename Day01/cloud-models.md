# Cloud Service Models

## Introduction

Cloud providers offer different service models based on how much responsibility the customer wants to manage. The three main service models are IaaS, PaaS, and SaaS.

## IaaS (Infrastructure as a Service)

In IaaS, the cloud provider offers virtual machines, storage, and networking. The customer is responsible for managing the operating system, applications, and middleware.

### Examples

- Virtual machines
- Storage
- Networking

## PaaS (Platform as a Service)

In PaaS, the cloud provider offers a platform for developing, running, and managing applications. The customer is responsible for the application and data, but the provider manages the underlying infrastructure.

### Examples

- Web applications
- Mobile applications
- APIs

## SaaS (Software as a Service)

In SaaS, the cloud provider offers complete applications over the internet. The customer only needs to use the application, while the provider manages everything else.

### Examples

- Email services
- Office applications
- Customer relationship management (CRM)
